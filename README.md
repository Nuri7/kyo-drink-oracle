# KYŌ KLUB - Drink Oracle

A beautiful, mobile-first PWA quiz application to help users discover their perfect matcha or hojicha drink at KYŌ KLUB. Built with standard HTML, CSS, and Vanilla JS for screaming fast performance and seamless offline capabilities.

## 📊 Analytics & Stats

This application uses PostHog for tracking user sessions, button clicks, pageviews, and session replays.

- **PostHog Dashboard:** [View Live Analytics & Sessions Here](https://eu.posthog.com/)

*(Ensure you are logged into your PostHog account to view the `Explore sessions` and general project dashboards).*

## 🛠 Technical Features

- **PWA Ready:** Installable directly on iOS/Android via `manifest.json` and internal Service Worker (`sw.js`).
- **PostHog Integrated:** Full tracking script implemented in UI routing logic.
- **Glassmorphism UI:** Custom CSS utilizing variable-driven modern web aesthetics.
- **Dynamic Quiz Engine:** Client-side Javascript logic for instant feedback and routing without server lag.
